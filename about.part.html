Über diese Seite

Alle Informationen über Mandate, Vertretungen, Links etc. werden automatisch aus der piratenmandate-Datenbank gezogen. Der einfachste Weg, eine Änderung zu melden ist ein pull request auf Github. Auch ohne Github-Account ist es nützlich, mir zu sagen, in welcher Zeile dieser Datei welche Änderung gemacht werden muss.

Der Code der Seite selbst ist im piratenmandate-frontend-Repository auf Github zu finden.

Quelle für die Daten auf dieser Seite ist neben eigener Recherche und Hinweisen die Seite „Mandate“ im Piratenwiki. Bisweilen können sich Informationen dort und hier widersprechen. Welche Quelle dann Recht hat, lässt sich pauschal nicht sagen.
