Über diese Seite

Alle Informationen über Mandate, Vertretungen, Links etc. werden automatisch aus der piratenmandate-Datenbank gezogen. Der einfachste Weg, eine Änderung zu melden ist ein pull request auf Github. Auch ohne Github-Account ist es nützlich, mir zu sagen, in welcher Zeile dieser Datei welche Änderung gemacht werden muss.

Der Code der Seite selbst ist im piratenmandate-frontend-Repository auf Github zu finden.

Die wichtigsten Quellen für die Informationen auf dieser Seite sind eigene Recherchen in Wahlergebnissen und Pressemitteilungen, Hinweise von anderen Piraten sowie die Seite „Mandate“ im Piratenwiki. Bisweilen können sich diese Seite und das Piratenwiki widersprechen – welche Quelle dann eher richtig liegt, lässt sich pauschal nicht sagen. Im Zweifel zählt für diese Seite das, was auf offiziellen Webseiten (etwa dem Ratsinformationssystem eines Gemeinderats), in Presseberichten oder in Pressemitteilungen von Gliederungen der Piratenpartei steht.
